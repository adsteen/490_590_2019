Lecture notes: Intro to git

2019-10-15

Resources

The resources I’m listing are better than any notes I would post, so I refer you to them for this lesson.

  • Software Carpentry’s git lesson
  • As a reminder, Software Carpentry’s the Unix (bash) shell lesson
  • Jenny Bryan’s extensive Happy git with R guide, which contains a lot of practical detail about setting up git and connecting it with GitHub. Some particularly important sections:
    • Philosophy and practical advice on git clients (like Git Kraken, which I’m using)
    • Connecting to GitHub
    • Setting up credentials for HTTPS so you can connect automatically
  • GitHub’s list of guides to get started with git and GitHub.

Looking forward

Our next lesson set is about shiny, a system to make graphical user interfaces (GUIs) in R. Many of you will be using shiny for your final projects.

If you’d like to include shiny in your final project, I recommend looking at Hadley Wickham’s early draft of a shiny textbook, as well as the RStudio’s shiny tutorial to get a sense of what shiny is and how it works.
